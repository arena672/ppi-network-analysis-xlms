# Protein-Protein Interaction (PPI) Network Analysis using XL-MS

## Overview
This project focuses on analyzing protein-protein interactions (PPIs) using cross-linking mass spectrometry (XL-MS) data, inspired by recent research in in vivo cross-linking studies.

## Objective
- Convert raw mass spectrometry data into MGF format
- Perform database search against Homo sapiens protein database
- Identify cross-linked peptides
- Analyze and interpret PPI networks

## Workflow
1. Raw MS Data Processing
2. Conversion to MGF (ProteoWizard)
3. Database Search (Homo sapiens - SwissProt)
4. Cross-link Identification
5. PPI Network Analysis

## Methodology (Based on Paper)
- MGF conversion using MSConvert
- Database search with defined mass tolerances
- Identification of cross-linked peptides
- Validation using scoring and filtering

## Tools Used
- ProteoWizard (MSConvert)
- Protein databases (SwissProt - Homo sapiens)
- Bioinformatics tools for analysis

## Practical Implementation
As part of this project, cross-linked peptide identification was performed using Protein Prospector.

- MGF files generated from raw MS data were used as input
- Database search was performed against Homo sapiens protein database
- Cross-linked peptides were identified based on scoring and filtering criteria

This step replicates key aspects of XL-MS based PPI analysis workflows used in research.
