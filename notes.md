# Notes on PPI Analysis using XL-MS

## What is PPI?
Protein-Protein Interaction (PPI) refers to physical interactions between proteins that are essential for biological processes.

## What is XL-MS?
Cross-linking Mass Spectrometry (XL-MS) is a technique used to identify interactions by chemically linking nearby residues and analyzing them via mass spectrometry.

## Key Steps in Workflow
- Cross-link proteins
- Digest into peptides
- Perform MS analysis
- Convert raw data to MGF format
- Perform database search
- Identify cross-linked peptides
- Construct PPI networks

## Key Insight
Cross-linking data helps identify which proteins are physically close, enabling detection of protein-protein interactions.
